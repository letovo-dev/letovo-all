# AGENTS.md

Instructions for agents working in this repository.

## Adding Changes

For every code, configuration, documentation, or submodule change:

1. Create a pull request for the change.
2. Request and wait for review from `ya-yara`.
3. Address review comments if any, then wait for `ya-yara` to approve the updated PR.
4. Wait for CI to finish successfully.
5. Only after the PR is approved by `ya-yara` and CI is green may you tell the user that the work is done.

Do not report a change as complete while the PR is still unreviewed, waiting for requested fixes, or has pending/failing CI.
